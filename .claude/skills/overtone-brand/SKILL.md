# Amplify Overtone Brand

Amplify Overtone is an open-source product under the **nxsflow** brand family. nxsflow (always lowercase) is the parent brand of NexusFlow UG (haftungsbeschränkt), positioned as "precise tools for technical professionals."

Overtone lives at `overtone.nxsflow.com`.

## Logo

### Tuning Fork Icon

The Amplify Overtone logo is a **hand-drawn tuning fork** — bold prongs with U-bend, handle, and resonance waves. Rendered in lavender (`#A78BFA`).

| Variant    | When to use                  |
| ---------- | ---------------------------- |
| Full       | With waves — large sizes     |
| Simplified | No waves — favicons, small   |

### Logo Rules

- Clear space: 1x icon height on all sides
- Minimum size: 24px digital, 10mm print
- Variants: full color, monochrome (`#1C1C1C`), reversed
- **Never** stretch, rotate, or recolor outside palette

---

## Design Language — "Open Workshop"

- Lavender with gradient treatments (lavender-to-rose)
- Documentation-style layouts
- Communicates: **collaboration, community, inviting contributions**

---

## Tone of Voice

**Inclusive, technical, community-minded.**

| Do                                               | Don't                                      |
| ------------------------------------------------ | ------------------------------------------ |
| Use "we" not "I"                                 | Use first-person singular                  |
| Write in documentation voice                     | Write in marketing voice                   |
| Welcome all skill levels                         | Assume advanced knowledge without context  |
| Be technically precise                           | Use hype or buzzwords                      |
| Invite contribution and collaboration            | Be gatekeeping or exclusionary             |

### Examples

- "We provide sync primitives that work offline-first."
- "Contributions are welcome — see the guide to get started."
- "This module extends Amplify Gen 2 with email and collaboration support."

### Parent Brand Baseline

nxsflow overall is **confident, clear, understated** — "We build precise tools." No hype, no buzzwords. Overtone inherits this foundation and layers on its community-oriented, inclusive voice.
